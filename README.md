# Employee Management System

A full-stack Employee Management application built using Spring Boot and Angular.

## Tech Stack

### Backend
- Java 17
- Spring Boot
- REST APIs

### Frontend
- Angular
- Bootstrap
- Reactive Forms

## Features

- Add Employee
- Update Employee
- Delete Employee
- View Employee
- Search Employees

## Backend

```bash
cd employee-backend
mvn spring-boot:run
```

Runs on:

```
http://localhost:8080
```

## Frontend

```bash
cd employee-frontend
npm install
ng serve
```

Runs on:

```
http://localhost:4200
```
